## 💻 Sobre o projeto

Este projeto consiste no desenvolvimento de um chatbot de suporte web utilizando a biblioteca Flask em Python. O principal objetivo foi aprofundar meus estudos no desenvolvimento de aplicações web,
compreendendo melhor o funcionamento de rotas, requisições HTTP (GET e POST), renderização de templates HTML e integração com banco de dados.

O sistema simula um atendimento automático, no qual o bot inicia a conversa assim que o usuário acessa o site e responde às dúvidas com base em opções pré-definidas, como horário de atendimento,
preços e suporte. Todas as mensagens são armazenadas em banco de dados SQLite, permitindo o registro do histórico da conversa.

Este projeto foi desenvolvido como prática de aprendizado, com foco em entender a biblioteca Flask e explorar conceitos relacionados à criação de bots de suporte e automação de atendimento,
além da comunicação entre backend e frontend.
